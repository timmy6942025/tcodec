# Golden Corpus

This directory contains procedurally generated test clips and encoded
reference bitstreams for regression testing.

## Regeneration

All files here are generated by `tools/gen_golden.sh`:

```bash
./tools/gen_golden.sh golden/
```

The YUV source clips and encoded bitstreams (`.tcv`) are **not committed**
to git — they are gitignored and must be regenerated locally.

Only `MANIFEST.sha256` is committed. After regeneration, verify integrity:

```bash
cd golden/ && sha256sum -c MANIFEST.sha256
```

## Contents

| File | Description |
|------|-------------|
| `*_128x128.yuv` | 128×128 YUV 4:2:0 source clips (gradient, checkerboard, noise, etc.) |
| `*_320x240.yuv` | 320×240 YUV 4:2:0 source clips |
| `*_96x80.yuv` | 96×80 YUV 4:2:0 source clips (non-CTU-aligned) |
| `*_qp*.tcv` | Encoded TCodec bitstreams at various QPs |
| `MANIFEST.sha256` | SHA256 hashes of all generated files |
